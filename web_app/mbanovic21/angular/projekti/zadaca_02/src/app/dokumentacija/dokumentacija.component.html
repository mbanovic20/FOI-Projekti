NAPOMENA
1. Prilikom pokretanja angular projekta u direktoriju "mbanovic21/angular" potrebno je najprije u terminalu pokrenuti naredbu "npm install" kako bi se instalirali svi potrebni node moduli. Nakon toga moguće je sa npm start pokrenuti angular projekt.
2. Prije pokretanja poslužiteljske strane na putanji "mbanovic21/server" pokrenite u terminalu "npm run pripremi" kako bi se instalirali node moduli koji su potrebni za rad.
3. Ne zaboravite prije pokretanja servera u konfiguracijsku datoteku "rwa_mbanovic21_conf.csv" unijeti svoje tmdbApiKeyV3 i tmdbApiKeyV4.
4. Kod pokretanja poslužiteljske strane "mbanovic21/server" sa npm start se pokreće glavni server "appServer.mjs". Prilikom interakcije sa bazom potrebno je pokrenuti i restServer.mjs, on se nalazi na putanji "mbanovic21/server/aplikacija/servis", te ondje pokrenuti naredbu "npm start" kako bi se pokrenuo REST server.
5. Kad se prijavite kao gost, i pokušate pristupiti nekoj od stranica kojima gost nema pristup, automatski će vas redirectati na početnu stranicu (ima ispis svake rute kojoj pristupate u konzoli). Tako da ne mislite da nije funkcionalno, već se trebate prijaviti sa pravom ulogom kako biste omogućili pristup određenim rutama.
